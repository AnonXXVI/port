Project Overview

This project is a specialized real estate platform designed for the Lesotho market, enabling landlords and real estate agents to post properties for sale or rent to both local and international buyers. To ensure trust and security, the platform leverages blockchain technology using smart contracts to verify the authenticity and legitimacy of each property listing. This helps prevent fraudulent activities and builds user confidence. Additionally, the platform incorporates self-sovereign identity technology, allowing users to maintain control over their digital identities without reliance on centralized authorities. To facilitate transactions, a Basotho-specific payment gateway is integrated, allowing for seamless payments in local currency. This project aims to transform the real estate landscape in Lesotho by providing a secure, transparent, and user-friendly platform for property transactions.
